# Improved fan duct (Waggster-compatible) — source

Contingency part: an improved part-cooling duct by **3DPrintBeginner**, designed so it
does **not** interfere with the Waggster BLTouch mount and cools from ~180° instead of
one side. Print this only if a test-fit shows the BLTouch mount fouling the stock duct.

Binary STL is **not** vendored — Printables/Thingiverse require a session for download.
